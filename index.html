template

A base template for starting projects. Features:

  • a full CSS library with reset, form and typographic base styles, and useful layout classes.
  • useful JS and jQuery extensions including transition class handling and form validation.
  • HTML5 template.

Files are designed to be concatenated and compressed.

All files are ignored unless specifically included in .gitignore (making it is safe to commit to template when the project has already added other files).

Major parts of this template are purloined from other sources. Credit is given in comments in the code.
